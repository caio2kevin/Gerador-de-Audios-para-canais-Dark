# 🎙️ Dark Voice Cloner

Gerador de áudio com síntese de voz por IA, 100% gratuito e local.
Sem APIs pagas, sem limites de créditos, sem internet após instalação.

## ✨ Funcionalidades
- Interface web dark com visual profissional
- 9 vozes pré-treinadas em Português, Inglês e Espanhol
- Controle de velocidade da fala
- Download do áudio gerado em .WAV
- Funciona 100% offline após instalação
- Roda no navegador via localhost

## 🛠️ Stack
- Python 3.11
- Flask (servidor web local)
- Kokoro ONNX 0.5.0 (engine TTS)
- HTML/CSS/JS (interface)

## ⚡ Como usar
1. Execute instalar.bat (Windows)
2. Execute iniciar.bat
3. Acesse http://localhost:5000

## 📋 Requisitos
- Python 3.11
- Windows 10/11
- ~500MB de espaço em disco
